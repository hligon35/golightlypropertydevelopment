# Golightly Property Development

A responsive, multi-page marketing website for Golightly Property Development in West Paducah, Kentucky.

## Pages

- Home
- About
- Services
- Projects
- Contact / project inquiry

## Run locally

No build step is required. Open `index.html` directly or run any static server, for example:

```bash
npx serve .
```

## Deployment

The site is compatible with GitHub Pages, Cloudflare Pages, Netlify, Vercel, and other static hosts. Set the publish directory to the repository root.
